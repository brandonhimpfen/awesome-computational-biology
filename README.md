## Genomics & Sequence Analysis

Tools and platforms for analyzing DNA, RNA, and genomic data.

- [BLAST](https://blast.ncbi.nlm.nih.gov/Blast.cgi) — Sequence alignment tool for comparing nucleotide and protein sequences.
- [BWA](http://bio-bwa.sourceforge.net/) — Burrows-Wheeler Aligner for mapping sequencing reads.
- [Bowtie](http://bowtie-bio.sourceforge.net/index.shtml) — Fast aligner for short DNA sequences.
- [SAMtools](http://www.htslib.org/) — Tools for manipulating sequencing data in SAM/BAM formats.

## Systems Biology & Modeling

Frameworks and tools for modeling biological systems and interactions.

- [COPASI](https://copasi.org/) — Software for simulation and analysis of biochemical networks.
- [CellDesigner](http://www.celldesigner.org/) — Modeling tool for gene-regulatory and biochemical networks.
- [SBML (Systems Biology Markup Language)](http://sbml.org/) — Standard format for representing computational models in biology.

## Structural Biology & Molecular Simulation

Tools for studying molecular structures and simulating biological systems.

- [GROMACS](http://www.gromacs.org/) — Molecular dynamics simulation package.
- [NAMD](https://www.ks.uiuc.edu/Research/namd/) — High-performance molecular dynamics simulator.
- [PyMOL](https://pymol.org/) — Visualization system for molecular structures.

## Machine Learning in Biology

Libraries and platforms applying AI and machine learning to biological data.

- [DeepMind AlphaFold](https://alphafold.ebi.ac.uk/) — AI system for predicting protein structures.
- [DeepChem](https://deepchem.io/) — Machine learning library for chemistry and biology.
- [scikit-learn](https://scikit-learn.org/) — General machine learning library used in bioinformatics workflows.

## Data Analysis & Visualization

Tools for analyzing and visualizing biological datasets.

- [Galaxy](https://usegalaxy.org/) — Web-based platform for accessible bioinformatics analysis.
- [IGV (Integrative Genomics Viewer)](https://software.broadinstitute.org/software/igv/) — Visualization tool for genomic data.
- [Plotly](https://plotly.com/) — Visualization library used in scientific computing.

## Databases & Repositories

Public repositories for biological data and computational biology research.

- [NCBI](https://www.ncbi.nlm.nih.gov/) — Database of genomic and biomedical information.
- [Ensembl](https://www.ensembl.org/) — Genome database for multiple species.
- [Protein Data Bank (PDB)](https://www.rcsb.org/) — Repository for 3D structural data of biological molecules.

## Workflows & Pipelines

Tools for building scalable and reproducible computational biology workflows.

- [Nextflow](https://www.nextflow.io/) — Workflow manager for scalable bioinformatics pipelines.
- [Snakemake](https://snakemake.readthedocs.io/) — Workflow engine for reproducible analysis.
- [Cromwell (WDL)](https://cromwell.readthedocs.io/) — Workflow execution engine for genomics pipelines.

## Education & Resources

Courses, tutorials, and platforms for learning computational biology.

- [Coursera Computational Biology](https://www.coursera.org/) — Courses covering computational biology and bioinformatics topics.
- [edX Computational Biology](https://www.edx.org/) — University-level courses in computational biology.
- [Rosalind](https://rosalind.info/) — Problem-solving platform for bioinformatics and computational biology.
